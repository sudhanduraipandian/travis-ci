# Addon Coverity Scan



## Types

* Hash
* Boolean



## Examples

```yaml
coverity_scan:
  build_script_url: string
  branch_pattern: string
  notification_email: string
  build_command: string
  build_command_prepend: string
```

```yaml
coverity_scan:
  project:
    name: string
    version: string
    description: string
```

```yaml
coverity_scan:
  project: string
```

```yaml
coverity_scan: true

```
