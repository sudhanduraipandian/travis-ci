# Deploy Chef Supermarket



## Type

Any of:

* Map (Hash)
* Enum (known String)

## Flags

None.


## Examples

```yaml
deploy:
  provider: chef_supermarket
  on: {}
  allow_failure: true
  skip_cleanup: true
  edge: true
  user_id: 
  client_key: 
  cookbook_category: string
```

```yaml
deploy: chef_supermarket

```
